###imap常用命令

####登陆: A LOGIN USER PASS
A是代号可以随便写 LOGIN是登录的意思，USER是用户名，PASS是密码。

####查询：A LIST “” *

####选择 ：A SELECT INBOX
这里选择的是收件箱，如果是想选择草稿箱的话就把INBOX改成Drafts即可

####查看邮件：A fetch 1 body[text]
1代表的是第一封邮件，text代表的是整个邮件整体，body[HEADER]代表的是返回完整的文件头信息。
